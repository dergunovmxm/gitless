Название : Php form registration

ТЗ: Требуется реализовать форму регистрации и авторизации

1. Необходимо создать форму со всеми возможными элементами:
    - input с типами - email, text, checkbox, file, radio, reset
    - textarea
    - select
    - button

2. Добавить проверку обязательных полей формы.

3. Подготовить таблицу и сохранить результаты отправки формы в базу данных.

4. Создать страницу со списком результатов.

5. Дополнительный функционал:
- форма должны быть создана используя bootstrap 4
- необходимо отправлять результаты на электронную почту администратора
- добавить функционал постраничной навигации для списка с результатами

Интерфейс : проект содержить форму с различными полями ввода данных и кнопками, с функиями "регистрация", "авторизация", "сброс".

1 - представляет собой форму регистрации http://site.devel/index.php

	Поля ввода данных

	name - ввод имени пользователя
    	surname - ввод фамилии пользователя
    	education - выбор образования (нет/школа/вуз)
    	role - выбор роли (ученик/преподаватель)
    	city - выбор города из выпадающего списка
    	password - ввод пароля
    	email - ввод почты
    	text - ввод в текстареа доп инофрмации
 	$uploadname - определение имени загружаемого файла 
    	$uploadpath - определение пути загружаемого файлла


	Кнопки

	- Регистрация- отправление данные в бд
	- Авторизация - перенаправление пользователя на страницу авторизации
	- Сброс - очистка всех полей в форме

После отправки формы, открывается страница со списком пользователей.

2 - форма авторизации на сайт для дальнейшего отображения зарегистрированных пользователей http://site.devel/site/index.php?mode=auth
    (так же в конце формы в index.pgp есть кнопка "Sign in" которая перенаправляет пользователя на страницу авторизации)

	Поле/я ввода данных
	
	- Эл. почта - поле для ввода почты пользователя(обязательное, уникальное)
	- Пароль - поле для ввода пароля пользователя(обязательно, проверяется по почте)

	Кнопка/и
	
	- Авторизация - кнопка подтверждения и отправки данных в бд с целью проверки их на подлинность

3- форма администратора, на ней можно посмотреть всех зарегистрированных пользователей.

Как запустить: распаковать архив site_for_simtech.rar, поместить полученные файлы на локальный сервер. Загрузить базу данных project_d.sql в phpmyadmin. 

4 - На оснвной странице после регистрации помещена кнопка "Связаться с нами", которая ведет на форму заявки. После заполнения формы, заявка отправляется на сервер. Список заявок отображается
в таблице "Список заявок"

Проект содерджит файл connect.php, где содержится подключение к базе данных. Данный файл подключен к файлам, где необходима работа с БД
